Quick Start

Clone o repositório

git clone https://github.com/euvanessa-prado/arquitetura_zero_etl.git

Crie o bucket para Terraform State

aws s3 mb s3://terraform-state-data-handson-mds-dev --region us-east-1

Provisione a infraestrutura

cd terraform/infra && terraform init && terraform apply

Carregue os dados

python code/insert_data_postgres/upload_to_s3.py

Verifique a replicação

python code/insert_data_postgres/test_redshift_connection.py

Fluxo de Dados

  • Ingestão: CSVs do MovieLens são enviados para o S3
  • Carga: Dados são inseridos no Aurora PostgreSQL
  • Replicação: Zero-ETL replica automaticamente para Redshift
  • Transformação: dbt cria modelos analíticos
  • Visualização: Metabase consome os dados transformados

Segurança

Credenciais no AWS Secrets Manager
RDS/Redshift em subnets privadas
Encryption em repouso
Acesso via SSM (sem SSH exposto)
Dados sensíveis removidos do código
Security Groups restritivos
